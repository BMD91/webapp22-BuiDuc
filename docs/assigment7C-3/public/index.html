This app, which is accompanied by a tutorial, is the 3rd example app of our incremental sequence of six examples of Plain JavaScript and Firebase Web Apps. This app supports the following operations:

  • Create a new order record
  • Retrieve and list all order records
  • Update a order record
  • Delete a order record
    • For purpose of the assigment 7-3, there are 4 additional actions connected to new class called Product. This will allow to test enumerations
  • Create a new product record
  • Retrieve and list all product records
  • Update a product record
  • Delete a product record
